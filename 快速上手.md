# Omni-Writer AI 快速上手（3 分钟）

> 不会编程也能用。跟着做就行。

---

## 第一步：装 Python（只需一次）

1. 打开 https://www.python.org/downloads/
2. 点黄色大按钮下载
3. 安装时 **务必勾选 "Add Python to PATH"**（很重要！）
4. 一路下一步装完

## 第二步：拿 API Key（免费）

1. 打开 https://platform.deepseek.com/api_keys
2. 注册账号（送 500 万 tokens，能写几十万字）
3. 点"创建 API Key"，复制下来（`sk-` 开头的一串）

## 第三步：安装本项目

双击 **`install.bat`**，粘贴刚才的 API Key，回车。完事。

## 第四步：开始写小说

双击 **`start.bat`**：

- 选 **1** → 浏览器自动打开网页控制台（推荐）
- 选 **3** → 不花 Key，先看看界面长什么样（Demo 模式）

然后：

1. 在"创世引擎"输入一句话灵感，比如：
   > 修仙者穿越到赛博朋克世界，发现天道是个可以被黑的系统
2. 点"开始创世"，AI 自动生成 3 卷大纲
3. 切到"创作流水线"，点"生成下一章"
4. 看着日志滚动：检索素材 → 写初稿 → 雷达扫描毒点 → 脱敏去 AI 味
5. 一章就出炉了。不满意？复制一段到"局部重绘"，让它重写

## 常用操作

| 我想... | 怎么做 |
|---|---|
| 写错了重来 | "创作流水线" → 章节回退 |
| 3 卷写完了 | "创世引擎" → 续写下一卷大纲 |
| 导出成小说文件 | "导出" Tab → 选 TXT → 导出全书 |
| 同时写两本书 | 顶部下拉切换书籍 |
| 看之前写的章节 | "章节浏览" Tab |

## 常见问题

**Q: 提示"未检测到 Python"？**
A: 装 Python 时没勾 "Add Python to PATH"，卸载重装记得勾上。

**Q: API Key 错了怎么办？**
A: 删掉 `config.yaml`，重新双击 `install.bat`。

**Q: 会花很多钱吗？**
A: DeepSeek 送的 500 万 tokens 大约能写 30-50 万字正文。之后充值约 1 元/万字，网文圈最便宜。

**Q: 生成一章要多久？**
A: 约 1-2 分钟（要经过 4 道 AI 工序，值得等）。
